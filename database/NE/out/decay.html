Frequently Useful Nuclear Decay Data File

Nuclear decay data and scheme of the frequently useful radioactivity nuclides can be obtained at this homepage. There are five retrieval ways: only for mass number, only for element, only for decay mode, a nuclide and a nuclide with decay mode.

The relevant data are based on the evaluated data up to August, 2000. Information on each nuclide contains half-life, nuclear decay data and its scheme.

¡¡

Retrieval Parameters

Mass Number: (the nucleus mass number of parent nuclide. example:22)
Element: (the element of parent nuclide. example:NA,11)
Decay Mode:
State (0:Ground State; 1 or 2: Isomer State)
