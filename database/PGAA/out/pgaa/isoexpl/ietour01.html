Tour of Isotope Explorer











Isotope Explorer (previously named VuENSDF) is a very 
general database/display program for nuclear 
structure and radioactive decay data.


The user can retrieve, select and display data from ENSDF 
(structure and decay data) and NSR (literature references) 
in several ways.



New features:

    

  • Nuclear chart generation including user interface to data

  • Band drawings

  • Plot function

  • Script language for retrieval and calculations
